Cardiac Signal Processing - Writeup

This is the Technical Writeup for the project

Introduction

This project is on the subject of methods used to measure cardiac signals.

Currently it consists of two major methods, the FFT and the STFT, with the FTFFT and arrythmia detection code currently in the works.

Scope

The scope of this project, or rather its current reason for existing is fairly simple, that being that we want to explore the two major methods used in the field to detect cardiac signals (FFT & STFT) and find a way to detect Arrythmia via some form of code - I should also note, that the FTFFT is a stretch goal as of current.
